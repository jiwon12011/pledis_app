# Generated Asset Direction

- Characters are separate transparent PNG layers.
- Character generation source should use a flat `#00ff00` chroma-key background, then remove it locally.
- Backgrounds must not include people.
- Mix aspect ratios by scene: intimate/mobile scenes use `9:16`, wide movement scenes use `16:9`, top-view scenes use `1:1`.
- Visual mood: 청량, 청순, low-saturation summer, natural hair, white/powder-blue/pale-denim outfits.
